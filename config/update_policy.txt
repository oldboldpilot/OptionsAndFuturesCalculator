   - Compile with clang-tidy and clang-format
   - Zero warnings, zero errors required
   - Command: `SKIP_CLANG_TIDY=1 ninja -C build bigbrother`

2. **Test Suite**
   - All tests must pass before pushing
   - Command: `ctest --test-dir build`
   - Review code for adherence to coding standards

3. **Documentation Updates** (Required for every push)
